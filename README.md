# Tree Todo App

A tree-structured todo list application with drag-and-drop, priority management, and full CRUD functionality.

## Tech Stack

- Frontend: React 18 + TypeScript + Vite + Ant Design
- Backend: Node.js + Express + TypeScript
- Database: SQLite + Prisma ORM
- Drag and Drop: @dnd-kit

## Features

- Tree-structured todo management with parent-child relationships
- Drag-and-drop reordering within the same level
- Move todos between parents
- Priority levels (High, Medium, Low) with visual indicators
- Mark todos as complete/incomplete
- Responsive dark theme design with modern aesthetics
- Smooth animations and micro-interactions

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Production

```bash
npm start
```

## Design

This app features a neo-brutalist aesthetic with:
- Dark theme with gradient mesh backgrounds
- Glassmorphism effects
- Custom typography (Space Mono + Outfit fonts)
- Smooth animations and hover effects
- Responsive design
